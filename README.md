# Hex-Game
Hex Game written with C++ language
Initially, the user determines the size of the game board starting from 5x5 (like 6x6,7x7 ...). He then chooses whether to play against the computer or with another user.

The user can save the game he / she played in the game, load and play another game he played previously, open a new game by changing the size of the game board, or see the last move.

For any player to win the game;
1) X expressions must reach from the first line to the last line.
2) O statements must reach the last column from the first column.

As users get closer to their goals, their scores increase.
